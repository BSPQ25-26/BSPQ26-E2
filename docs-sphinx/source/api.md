# API Documentation

The REST API is documented through Springdoc OpenAPI in the running application.
Use the following endpoints after starting the backend:

- Swagger UI: `http://localhost:8080/swagger-ui.html`
- OpenAPI JSON: `http://localhost:8080/api-docs`

## Main API Areas

- **Authentication**: `/api/auth`
- **Movies**: `/api/movies`
- **User Movie Status**: `/api/users/{userId}/movies`
