# Overview

MovieTrakk is a Spring Boot web application for tracking movies.
It combines a Java backend with a lightweight frontend served from `src/main/resources/static`.

The site provides:

- user registration, login, and role-based access control
- movie catalog browsing and filters
- watch-later, watched, liked, and disliked lists
- notes per movie
- admin operations for creating, updating, and deleting movies

The documentation portal is built with Sphinx and published automatically through GitHub Actions.
